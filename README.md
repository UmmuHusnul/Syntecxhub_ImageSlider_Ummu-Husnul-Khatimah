# Project 2 - Image Slider / Carousel

A responsive and interactive Image Slider (Carousel) built using HTML, CSS, and JavaScript.  
This project allows users to navigate through images using next/previous buttons, autoplay functionality, and slide indicators.

## 📌 Features

- ⏮ Previous button navigation  
- ⏭ Next button navigation  
- ▶ Auto-play functionality using JavaScript intervals  
- 🔘 Slide indicators (dots)  
- 🎞 Smooth transition animations between slides  
- 📱 Fully responsive and mobile-friendly design  
- ✨ Clean and modern UI  

---

## ⚙️ How It Works

- Images are displayed one at a time.
- Users can click **Next** or **Previous** to change slides.
- The slider automatically changes slides using `setInterval()`.
- Clicking on indicator dots jumps directly to a specific slide.
- Smooth CSS transitions create a sliding animation effect.
- The slider adapts to different screen sizes using responsive CSS.
